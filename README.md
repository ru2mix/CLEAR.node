Локальный сервер для менеджера подключений. Работает на SQLite.
Первый авторизовавшийся пользователь на сервере, будет админом.

https://clearbat.iiko.online/

Настройка поправить в main.py ip:
    uvicorn.run("main:app", host="0.0.0.0", port=8001, reload=False)
Заменить на свой. Можно прописать 0.0.0.0, порт указываем любой.

В config.py указаны данные для авторизации на главном сервере, под общей УЗ CLEAR.bat, и название локальной БД.
